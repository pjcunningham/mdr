# Markdown Reader
A basic Markdown reader built with [Flet](https://flet.dev/).

## Features
- Support for Dark Mode.
- Simple Markdown viewing.
- File picking.

## How to Run
To run the application, use the following command:
```bash
uv run python src/mdr/main.py
```

## How to Test
To run the tests, use:
```bash
uv run pytest
```
